# PenguinDive

The goal of PenguinDive is to dive into the Palmer penguins dataset,
aimed at teaching basic statistical concepts and ecological data
exploration.

## Installation

You can install the development version of PenguinDive like so:

``` r
# install.packages("pak")
# pak::pak("Vinanden/PenguinDive")

# or

# install.packages("remotes")
# remotes::install_github("Vinanden/penguinDive")
```

## Example

You can launch the PenguinDive app with:

``` r
library(PenguinDive)
run_app()
```
